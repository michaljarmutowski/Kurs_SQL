Zbiór ćwiczeń i zapytań SQL wykonanych podczas nauki MySQL i PostgreSQL.

MySQL:
-DDL – tworzenie i modyfikowanie tabel (CREATE, ALTER, DROP)
-DML – manipulacja danymi (INSERT, UPDATE, DELETE)
-SELECT – filtrowanie, sortowanie, grupowanie
-JOIN – łączenie tabel (INNER, LEFT, RIGHT JOIN)
-UNION – łączenie wyników zapytań

PostgreSQL:
-DDL / DML – tworzenie tabel, manipulacja danymi
-SELECT – zapytania i filtrowanie
-Wgrywanie danych – importowanie danych do bazy

Technologie:
-MySQL
-PostgreSQL
